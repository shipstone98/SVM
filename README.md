# SVM

An instruction-based virtual machine written in C#.

Code for the first piece of assessed coursework for the Components Based Architectures model on the MEng Computer Science course at the University of Hull.

## Notes
- Before a DLL can be loaded, you must add its hash to the assemblyHash.json file in the VirtualMachine directory
- Supported hashes include MD5, SHA1, SHA256, SHA384 and SHA512
- MD5 and SHA1 are advised against as they are widely regarded as insecure
